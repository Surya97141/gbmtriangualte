Binary Search on Answer variants:
Binary Search on Answer
Search for the optimal value in [lo, hi]
O(log(range) × isFeasible) ✓ 1.7M ops
· Minimize maximum or maximize minimum
· Answer is a value in a known range
· isFeasible(X) can be checked greedily in O(n)
Code template ▼ 
int lo = MIN_ANSWER, hi = MAX_ANSWER;
while (lo < hi) {
  int mid = lo + (hi - lo) / 2;
  if (isFeasible(mid)) hi = mid;
  else                 lo = mid + 1;
}
return lo;
⚠ isFeasible must be monotone — verify with 2 examples
⚠ Integer overflow: use lo + (hi-lo)/2 not (lo+hi)/2
