What does your DP state index represent?
What varies between different subproblems?
A single position in one sequence
dp[i] = answer for prefix ending at i
Two positions — one in each sequence
dp[i][j] = answer for prefixes of length i, j
A range [i, j] within one sequence
dp[i][j] = answer for subarray from i to j
A subset of elements
dp[mask] = answer for subset as bitmask
A node in a tree
dp[node] = answer for subtree at node
Position + remaining capacity
dp[i][w] = best value with capacity w
1D DP
State: dp[i] — maximum sum of subarray ending exactly at index i
Base: dp[0] = a[0]
Fill order: Left to right: i = 0, 1, 2, ..., n-1
Complexity: O(n) time → O(1) with rolling variables
⚠ Define dp[i] precisely — "ending AT i" vs "ending BY i" are different
Possible optimizations:
Rolling Array: O(n) → O(1) space Convex Hull Trick: O(n²) → O(n log n) if linear cost
